1. CSS选择器 - 如何选中HTML元素
元素选择器
通过HTML标签名选择元素

工作原理： CSS通过HTML标签名来选中页面上的所有同类型元素。

HTML结构：

普通div
普通div
这是p标签
这也是p标签
普通span
HTML代码：
<body>
  <div>普通div</div>
  <div>普通div</div>
  <p>这是p标签</p>
  <p>这也是p标签</p>
  <span>普通span</span>
</body>
CSS代码：
/* 选中所有p标签 */
p {
  color: #e74c3c;
  font-weight: bold;
}

/* 注意：div和span标签不会被选中 */
总结： 元素选择器会选中页面上所有匹配的HTML标签。上面的CSS代码会选中所有<p>标签，将它们变成红色和粗体，但不会影响<div>和<span>标签。
类选择器
通过class属性选择元素

工作原理： CSS通过HTML元素的class属性值来选中元素。一个元素可以有多个class，用空格分隔。

HTML结构：

普通div
class="highlight"的div
普通div
class="highlight highlight-bold"的div
class="highlight"的p标签
HTML代码：
<div>普通div</div>
<div class="highlight">class="highlight"的div</div>
<div>普通div</div>
<div class="highlight highlight-bold">class="highlight highlight-bold"的div</div>
<p class="highlight">class="highlight"的p标签</p>
CSS代码：
/* 选中所有class包含"highlight"的元素 */
.highlight {
  background-color: #3498db;
  color: white;
  padding: 10px;
}

/* 选中所有class包含"highlight-bold"的元素 */
.highlight-bold {
  font-weight: bold;
  background-color: #2ecc71;
  /* 会覆盖上面的背景色 */
}
总结： 类选择器以点号(.)开头，后面跟class名称。它可以选中所有具有该class的元素，无论是什么HTML标签。一个元素可以有多个class，样式会合并应用。
ID选择器
通过id属性选择元素

工作原理： CSS通过HTML元素的id属性值来选中元素。id在页面中应该是唯一的。

HTML结构：

我是主标题(id="main-title")
普通div
特殊盒子(id="special-box")
普通div
HTML代码：
<div id="main-title">我是主标题(id="main-title")</div>
<div>普通div</div>
<div id="special-box">特殊盒子(id="special-box")</div>
<div>普通div</div>
CSS代码：
/* 选中id为"main-title"的元素 */
#main-title {
  color: #e74c3c;
  font-size: 24px;
  font-weight: bold;
  text-align: center;
  border: 2px solid #e74c3c;
  padding: 10px;
}

/* 选中id为"special-box"的元素 */
#special-box {
  background-color: #2ecc71;
  color: white;
  padding: 15px;
  border-radius: 4px;
  text-align: center;
}
总结： ID选择器以井号(#)开头，后面跟id名称。它应该用于选择页面中唯一的元素。与class不同，一个id在页面中只能出现一次。
伪类选择器
根据元素状态选择元素

工作原理： 伪类选择器用于选择处于特定状态的元素，比如鼠标悬停、被点击等。

HTML结构：

将鼠标悬停在下面的盒子上：

悬停在这里
这是一个链接
HTML代码：
<div class="hover-box">悬停在这里</div>
<a href="#">这是一个链接</a>
CSS代码：
/* 基础样式 */
.hover-box {
  background-color: #9b59b6;
  color: white;
  padding: 15px;
  border-radius: 4px;
  text-align: center;
  cursor: pointer;
  transition: all 0.3s;
}

/* 鼠标悬停时的样式 */
.hover-box:hover {
  background-color: #8e44ad;
  transform: scale(1.05);
}

/* 链接基础样式 */
a {
  color: #3498db;
  text-decoration: none;
  display: inline-block;
  margin-top: 10px;
  padding: 5px 10px;
  border-radius: 4px;
}

/* 链接鼠标悬停时的样式 */
a:hover {
  background-color: #3498db;
  color: white;
}
总结： 伪类选择器以冒号(:)开头，用于选择元素的特殊状态。最常用的是:hover（鼠标悬停），还有:active（被点击时）、:focus（获得焦点时）等。
2. CSS盒子模型 - 理解元素布局
盒子模型结构
每个元素都是一个矩形盒子
外边距 margin: 30px margin
边框 border: 5px border
内边距 padding: 20px padding
内容 content width: 200px
height: 100px

这是实际内容

CSS代码：
/* 盒子模型CSS代码 */
.box {
  /* 内容区域大小 */
  width: 200px;
  height: 100px;
  
  /* 内边距：内容与边框的距离 */
  padding: 20px;
  
  /* 边框：盒子的边框线 */
  border: 5px solid #3498db;
  
  /* 外边距：盒子与其他元素的距离 */
  margin: 30px;
  
  /* 背景颜色，方便观察 */
  background-color: #f8f9fa;
}
盒子模型理解： 就像包装一个礼物，内容就是礼物本身，内边距是填充物，边框是包装盒，外边距是盒子与其他物品的间隔。
box-sizing属性
控制盒子尺寸计算方式

两种盒子模型对比：

content-box (默认)

元素总宽度 = 150 + 20×2 + 5×2 = 200px

border-box (推荐)

元素总宽度 = 150px (包含padding和border)
content-box 模型：
/* content-box：宽度只包含内容 */
.box1 {
  box-sizing: content-box; /* 默认值 */
  width: 150px;
  padding: 20px;
  border: 5px solid #3498db;
  /* 实际总宽度 = 150 + 20×2 + 5×2 = 200px */
}
border-box 模型：
/* border-box：宽度包含内容、内边距和边框 */
.box2 {
  box-sizing: border-box; /* 推荐使用 */
  width: 150px;
  padding: 20px;
  border: 5px solid #2ecc71;
  /* 实际总宽度 = 150px */
}
提示： 在实际项目中，通常会在CSS开头设置 * { box-sizing: border-box; }，这样更符合直觉，方便布局。

文字样式
控制文字外观

红色、24px、加粗、居中文字

蓝色、16px、斜体、下划线、1.8倍行高的文字。注意行与行之间的间距增大了，阅读更舒适。

display属性
控制元素显示方式

block（块级元素）: 独占一行，可设置宽高

块级元素1
块级元素2

inline（行内元素）: 不独占一行，不可设置宽高

行内元素1 行内元素2

inline-block（行内块元素）: 不独占一行，可设置宽高

行内块1
行内块2
CSS代码：
/* display属性CSS代码 */
/* 块级元素 */
div {
  display: block;
  width: 200px;
  height: 100px;
}

/* 行内元素 */
span {
  display: inline;
  /* 不能设置宽高 */
}

/* 行内块元素 */
.inline-block {
  display: inline-block;
  width: 100px;
  height: 50px;

Flex布局
现代布局方式 - 一维布局

什么是Flex布局？ Flex布局（弹性盒子布局）是CSS3中新增的一种布局方式，可以轻松实现各种复杂的页面布局，特别是在一维布局（水平或垂直方向）中表现出色。

核心概念： Flex布局中有两个重要概念：主轴（main axis）和交叉轴（cross axis）。默认情况下，主轴是水平的，交叉轴是垂直的。

1. Flex容器属性

flex-direction
设置主轴方向

属性值 效果
row (默认) 主轴为水平方向，起点在左端
row-reverse 主轴为水平方向，起点在右端
column 主轴为垂直方向，起点在上端
column-reverse 主轴为垂直方向，起点在下端
CSS代码：
.container {
  display: flex;

2. Flex项目属性

flex
定义项目的放大比例和缩小比例

flex属性是flex-grow, flex-shrink, flex-basis的简写

flex: 1
flex: 1
flex: 2 (占2份)

三个项目的比例是1:1:2，所以第三个项目占用的空间是前两个的2倍。

属性值 效果
flex: 1 简写形式，相当于 flex: 1 1 0%
flex: 0 0 200px 不放大，不缩小，固定宽度200px
flex-grow: 1 定义项目的放大比例，默认0（不放大）
flex-shrink: 0 定义项目的缩小比例，默认1（空间不足时缩小）
flex-basis: 200px 定义项目在主轴上的初始大小
CSS代码：
/* 容器 */
.container {
  display: flex;
}

/* 项目 */
.item1 {
  flex: 1; /* 占1份 */
}

.item2 {
  flex: 2; /* 占2份 */
}

.item3 {
  flex: 0 0 200px; /* 固定宽度200px */
}
align-self
单个项目在交叉轴上的对齐方式

align-self允许单个项目有与其他项目不一样的对齐方式

align-items: flex-start
align-self: center
align-self: flex-end
align-items: flex-start
CSS代码：
.container {
  display: flex;
  align-items: flex-start;
}

.item2 {
  align-self: center; /* 覆盖容器的align-items设置 */
}

.item3 {
  align-self: flex-end; /* 覆盖容器的align-items设置 */
}

Flex布局学习要点：
  • 先设置容器为 display: flex 开启Flex布局
  • 理解主轴和交叉轴的概念
  • justify-content 控制主轴对齐方式
  • align-items 控制交叉轴对齐方式
  • flex-direction 控制主轴方向
  • flex-wrap 控制是否换行（默认不换行）
  • 项目属性 flex 控制项目如何伸缩
  • 项目属性 align-self 允许单个项目有不同的对齐方式
